# 한국ITS학회 2025년도 춘계학술대회 학부논문경진대회

데이터는 '서울 열린 데이터 광장'에서 제공하는 '서울시 버스노선별 정류장별 시간대별 승하차 인원 정보' 데이터 중 2025년 1월, 2월 데이터를 사용합니다.
